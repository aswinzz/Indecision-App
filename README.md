A Todo app made using ReactJS.

# Install

* git clone https://github.com/aswinzz/Indecision-App.git
* `cd Indecision-App`
* run `yarn install`
* run `yarn run build`
* run `yarn run dev-server`
* open `localhost:8080` in your browser
